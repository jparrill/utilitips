- Execute commands from vim
```
:! pwd
:! mkdir path/to/folder
```

- Search and replace
```
:%s/word_to\ search/words\ to\ replace\ with/g
:%s/word_to\ search/words\ to\ replace\ with/
```

- Split
```
# Horizontal
:vspl

# Vertical
:spl
```

- Buffers
```
# Next Buff
:bn

# Previous Buff
:bp

# Delete Buff
:bd
```
